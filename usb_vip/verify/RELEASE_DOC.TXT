*System requirement*
1. VCS version is as least vcs-2015.x

*Enrironment structure*
                        serial
In house HOST VIP <-----------------> In house DEVICE VIP

*How to run*
1. Go to ./tb folder
2. Source the environment setting variable in brtvip/vips/brt_usb/verif
    source sim_setup.csh
4. Run simulation by choosing the test pattern and speed
    make TESTNAME=test_pattern_name SPEED=speed SEED=1

  + test_pattern_name: test name which included in brt_usb_test_pkg.sv
  + speed: choose hs, fs or ls
  + SEED: any number for generating random seed. Dont care if no random

    Ex: make TESTNAME=brt_usb_iso_in_mps77_vseq_test SPEED=hs SEED=1

*Support mode*
Mode 1: USB Host only
Mode 2: USB Host + Monitor
Mode 3: USB Device only
Mode 4: USB Device + Monitor
Mode 5: USB Monitor only

*How to instant this VIP to env*
refs: trunk/brtvip/vips/brt_usb/verif/sv/brt_usb_env.sv

Mode 1: USB Host only
a) Instance the host agent
b) Set USB interface to host agent
c) Parse config to host agent
d) Add callback for using error injection (optional)

Mode 2: USB Host + Monitor
a) Instance the host agent
b) Set USB interface to host agent
c) Parse config to host agent
d) Add callback for using error injection (optional)
e) Enable monior of host agent to check the incoming packet

Mode 3: USB Device only
a) Instance the device agent
b) Set USB interface to device agent
c) Parse config to device agent
d) Add callback for modifying transfer size and using error injection

Mode 4: USB Device + Monitor
a) Instance the device agent
b) Set USB interface to device agent
c) Parse config to device agent
d) Add callback for modifying transfer size and using error injection
e) Enable monior to check the incoming packet

Mode 5: USB Monitor only (use in case not inclusing host/device VIP)
a) Instance the monitor agent
b) Set USB interface to monitor agent
c) Create new config for monitor agent, use as device mode
d) Disable vip_enable and set mon_enable monior to check the incoming packet
